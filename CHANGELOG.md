# Changelog

## Version 1.0 (Initial Release)

**Developer:** yusufalvian16

### Fitur Utama
- ✅ Scan dan tampilkan media dari WhatsApp Status
- ✅ Download media (Gambar, Video, Audio, Dokumen)
- ✅ Preview media sebelum download
- ✅ Manajemen file yang sudah diunduh
- ✅ Bottom navigation dengan 3 tab (Home, Downloads, Settings)
- ✅ Material Design 3 UI
- ✅ Permission handling untuk berbagai Android version
- ✅ Tidak berjalan di background

### Teknologi
- Kotlin
- Jetpack Compose
- Material Design 3
- Coroutines & Flow
- Coil (Image Loading)
- Navigation Compose
- ViewModel & StateFlow

### Catatan
- Aplikasi ringan dan cepat
- Semua proses hanya terjadi saat aplikasi aktif
- File diunduh ke folder Downloads/WhatsAppDownloader
